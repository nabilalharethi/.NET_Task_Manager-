# Task Manager

A simple .NET console application for managing daily tasks.

## Features
- Add tasks
- View all tasks
- Mark tasks as complete
- Delete tasks

## Getting Started

### Prerequisites
- .NET 6.0 SDK or higher

### Build and Run
```bash
dotnet build
dotnet run --project src/TaskManager.App
```

## Project Structure
- `src/TaskManager.App/` - Main application
  - `Models/` - Data models
  - `Services/` - Business logic
